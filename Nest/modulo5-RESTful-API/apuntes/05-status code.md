- 1xx → Información
- 2xx → Ok
- 3xx → Redirección
- 4xx → Error del cliente
- 5xx → Error del servidor

- 200 → Success
- 201 → Created
- 301 → Moved Permanently
- 400 → Bad request
- 401 → Unauthorized
- 403 → Forbidden
- 404 → Not found
- 500 → Internal server error
